# Simple Contacts App

This is a simple contacts application built with Python for the backend and React with Vite for the frontend. It's designed to help you manage your contacts in an easy and intuitive way.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

What things you need to install the software and how to install them:

\```sh
pip install -r requirements.txt
npm install
\```

### Installing

1. Clone the repository:

```sh
git clone https://github.com/aaxyat/simplecontactsapp.git
```

2. Navigate to the project directory:

```sh
cd simplecontactsapp
```

3. Install the requirements:

```sh
cd backend
pip install -r requirements.txt
cd ..
cd frontend
npm install
```

4. Run the backend:

```sh
cd backend
python main.py
```

5. Run the frontend

```sh
cd backend
npm run dev
```
